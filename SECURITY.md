# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within GGNN 2025, please send an email to [student232433@ptnk.edu.vn](mailto:student232433@ptnk.edu.vn).

All security vulnerabilities will be promptly addressed.

## Security Measures

This repository implements the following security measures:

- **Secret Scanning**: Enabled - Detects accidentally committed secrets
- **Security Advisories**: Enabled - For disclosing security vulnerabilities
- **Dependency Review**: Dependencies are regularly reviewed for vulnerabilities

## Code Security

- No external API keys or secrets are stored in the codebase
- All model weights are publicly available and do not contain sensitive data
- The code does not collect or transmit user data

## Responsible Disclosure

We appreciate the security research community's efforts in helping keep our project secure. If you believe you have found a security vulnerability, please report it responsibly by:

1. Emailing the maintainer directly
2. Providing detailed steps to reproduce the issue
3. Allowing reasonable time for a fix before public disclosure

Thank you for helping keep GGNN 2025 and its users safe!
